## üöÄ How to Use

1.  Download `ForgeCommander.exe` from the **Releases** tab.
2.  **Right-click -> Run as Administrator** (Required for the macro to control the game).
3.  **Setup:**
    *   Go to **Configuration** tab.
    *   Set **Mining Position** and add **Targets** (hover over buttons in-game).
    *   (Optional) Add your Discord Webhook URL.
4.  **Start:**
    *   Go to **Dashboard** and click **ACTIVATE SYSTEM**.
